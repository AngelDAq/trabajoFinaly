Metodos existentes Windows

Herramientas integradas en cada sistema

Windows

Tasklist

Descripcion:Muestra todos los procesos en ejecucuion

Sintaxis
cdm

tasklist [opciones]

Opciones mas usadas

  • /v, Muestra detalles adicionales(PID, estado, CPU, etc)
  • /fi, "condicion" filtra resultados segun criterios
  • /svc, Muestra los servicios en ejecucion dentro de cada proceso
  • /m, Muestra los modulos (bibliotecas DLL) usados por cada proceso
  • /fo, Formato de salida

Ejemplo:

cmd

tasklist /o TABLE /fi "STATUS eq running"

Explicacion de ejemplo tasklist

Get-Process (PowerShell)

Descripcion:Lista los procesos en ejecucuon

Sintaxis:
powershell

Get-Process [nombre_proceso] [opciones]

Opciones mas usadas

  • -Name, Filtra por nombre
  • -Id, Filtra por ID del proceso
  • Format-Table -AutoSize presenta la informacion de forma legible

Ejemplo:

powershell

Get-Process -Name chrome

Explicacion de powerShell

WMIC

Descripcion:Proporciona informacion avanzada sobre procesos

Sintaxis
cdm

wmic process get [campo1, campo2]

Opciones mas usadas

  • where Filtra por condiciones
  • list full, muestra todos los detalles

Ejemplo:

cmd

wmic process where "name=`explorer.exe`" get ProcessId, ExecutablePath

Explicacion de wmic

Proceso de tasklist

opcion tasklist

Al Ingresar al cmd y poner este comando mostrara todos los procesos

opcion /v

Al Ingresar al cmd y poner este comando mostrara estados de CPU mas que todo detalles adicionales

opcion /m

Al Ingresar al cmd y poner este comando mostrara las bibliotecas usadas para cada proceso

opcion /svc

Muestra los servicios en ejecucion dentro de cada proceso
